📖 노션 연동 설정 가이드

1
인테그레이션 생성
notion.so/my-integrations 접속 → "+ New integration" → 이름 입력 → Submit
→ Internal Integration Token 복사
2
노션 데이터베이스 생성
노션에서 새 페이지 → "Database - Full page" 선택
아래 속성을 동일하게 추가:
Nametitle (기본)
날짜Date
시간Text
유형Select
우선순위Select
상태Select
태그Multi-select
메모Text
앱IDNumber
3
인테그레이션 연결
데이터베이스 페이지 우측 상단 ⋯ → "Connections" → 생성한 인테그레이션 추가
4
Database ID 확인
데이터베이스 URL에서 복사:
notion.so/여기가DB_ID?v=...
